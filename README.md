# 💰 ExpenseTracker 個人記帳系統

這是一個使用 C# Windows Forms 開發的個人記帳應用程式，結合 SQL Server LocalDB 進行資料儲存。系統提供直覺的操作介面、動態資料庫連動、以及即時的視覺化圓餅圖分析，幫助使用者輕鬆管理日常收支與多個錢包的財務狀況。

---

## 📌 初始畫面
系統啟動後，會呈現乾淨的儀表板介面，分為「圖表區」、「篩選及表格區」與「資料編輯區」。

## 🚀 基本功能

### 1. 新增資料 (Add)
* **步驟**：在左下方「操作功能」選擇「新增資料」 ➜ 於右側「帳目資料」填寫日期、類別、錢包、金額與備註 ➜ 勾選「支出」或「收入」 ➜ 按下 **[執行]**。
* **效果**：新增成功後會跳出確認視窗，按下確定後，中央的帳目表格會立刻顯示剛才的紀錄，左側的支出比例圓餅圖也會即時更新。

### 2. 修改資料 (Update)
* **步驟**：先在中央的「資料表格」點擊想要修改的那一筆帳目 ➜ 在左下方選擇「修改資料」 ➜ 於右側更新想要修改的欄位 ➜ 按下 **[執行]**。
* **效果**：執行後，表格與圓餅圖會立即根據最新的資料進行重繪與修正。

### 3. 刪除資料 (Delete)
* **步驟**：先在表格中點選要刪除的帳目（**注意：即使該筆資料目前已經是反白的藍色選取狀態，執行刪除前仍建議再次點擊確認選取**） ➜ 勾選「刪除資料」 ➜ 按下 **[執行]**。

## ✨ 進階功能

### 💼 動態錢包管理
為了讓使用者能精準追蹤各個銀行帳戶或支付工具的金流，系統導入了錢包功能：
* 下拉選單預設選項為「現金」。
* **動態新增**：若想使用其他錢包（如：信用卡、悠遊卡、特定銀行帳戶），不需額外設定，只需在輸入框「直接打字輸入」，系統存入後便會自動學習，日後篩選與記帳時皆可直接選取。

### 🔍 智慧交叉篩選
使用者可自由搭配上方選單進行資料過濾：
* 支援「日期範圍」、「錢包種類」、「帳目類別」三管齊下的交叉篩選。
* 選取條件後，下方的資料表格與左側的圓餅圖皆會 **「即時無縫更新」**，完全不需額外點擊搜尋按鈕。

### 📊 即時收支統計與視覺化
* **財務看板**：在表格下方會即時顯示目前篩選條件下的「總收入」、「總支出」與「總結餘」。
* **色彩防呆提示**：當總結餘為正數時，數字會呈現 **綠色**；若出現透支（負數）則會以 **紅色** 警示，讓使用者的財務狀況一目瞭然。
* **圓餅圖分析**：自動抓取當前篩選範圍內的各類別支出總額，繪製成精美的佔比圓餅圖，方便檢視消費習慣。
